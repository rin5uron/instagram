## Twitter → ClaudeCode → 台本生成の仕組み

### Phase 1: 手動運用（現在）
1. TwitterでAI系トレンドを手動収集
2. ClaudeCodeに投げて台本化
3. Instagram/YouTube投稿

### Phase 2: 半自動化
1. Twitterスクレイピング
2. ClaudeCode APIで台本自動生成
3. 人間がチェック→投稿

### Phase 3: 完全自動化
1. トレンド収集→台本生成→投稿まで自動化
2. この仕組み自体を商品化
