# Instagram運用プロジェクト

ニートな管理栄養士｜AI丸投げ美肌ごはん

## アカウントコンセプト

**世界観**: AI任せ / やる気ない / でも崩れない / 美肌は維持 / ゆるい

**投稿の軸**:
1. AIにごはん決めさせる
2. そのまま食べる
3. 管理栄養士的に軽く補足
4. "ちゃんとしてないのに整ってる"を見せる

## 運用フェーズ

### フェーズ1: 土台作り（現在）
- 動画投稿30本以上
- 型を固定（テンプレ化）
- 世界観の統一
- 信頼性の確立

### フェーズ2: 収益導線スタート
- アフィリエイトブログ開設
- インスタと連携
- アフィ案件を自然に挿入

### フェーズ3: 自動化・拡張
- 固定化アフィ獲得
- アフィリエイトデータベース活用
- アフィ自動化プロジェクト構築

### フェーズ4: 自社展開
- 自社商品開発（食品・美容系）
- アプリ開発
- 自社サービスへ誘導

## リポジトリ構成

```
├── templates/                      投稿テンプレート
│   ├── post-template.md           固定フォーマット
│   └── video-script-format.md     動画構成（30秒固定型）
├── scripts/                       台本置き場
│   ├── generate-script.md         台本生成ガイド
│   ├── 001.md                    台本1
│   ├── 002.md                    台本2
│   └── ...
└── README.md                      このファイル
```

## テンプレート

- [投稿フォーマット](templates/post-template.md) - キャプション構成と世界観
- [動画構成（30秒固定型）](templates/video-script-format.md) - タイムライン別の撮影内容

## 使い方

1. [動画構成テンプレート](templates/video-script-format.md) を確認
2. `scripts/generate-script.md` を参照して台本を作成
3. 生成した台本を `scripts/` に番号順で保存
4. 撮影・投稿

## Issue管理

思いついたアイデア、改善点、問題点などは全てIssueで管理:
- 投稿アイデア
- テンプレート改善案
- 運用上の課題
- フェーズ移行の検討事項
