**投稿日:** 2026-04-21

## 1. ChatGPT用プロンプト（スクショ用）

```
冷蔵庫にキャベツと豚肉と焼きそば麺しかない。
美肌に良い焼きそばのアレンジ教えて。
```

## 2. ChatGPT想定回答

**料理名:** ビタミンC増し増し美肌焼きそば

**材料:**
- 焼きそば麺 1袋
- キャベツ 1/4個
- 豚肉 100g
- レモン汁 大さじ1
- 付属のソース

**作り方:**
1. キャベツは大きめにちぎる（ビタミンC流出を防ぐ）
2. 豚肉を先に炒める（タンパク質でコラーゲン生成）
3. キャベツを加えて軽く炒める（加熱しすぎない）
4. 麺を入れてソースと絡める
5. 仕上げにレモン汁をかける（ビタミンC追加）

**調理ポイント:** キャベツは加熱しすぎないこと
**美肌・栄養ワード:** ビタミンC、コラーゲン生成、タンパク質

### 良かった点
- 賞味期限切れ焼きそばのリアル感が出た
- ChatGPT画面で具体的な質問を見せられた
- 「作る＋守る」のコピーが秀逸
- 「麺はすでにほぐれていた」のオチでゆるさが出た
- 3つのコツを明示して見やすい構成

### 気になった点
- 買い出しシーンを入れた（削る予定だったが残した）
- 編集時間が目標より長かった（特にテロップ作業）
- 初回のため試行錯誤が多かった
