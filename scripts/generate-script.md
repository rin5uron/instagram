# 台本生成ガイド

このファイルでは、Instagram投稿用の台本を生成する方法を説明します。

## Claude（AI）を使った台本生成

### 基本プロンプト

```
以下のアカウントコンセプトに基づいて、Instagram投稿用の台本を作成してください。

【アカウントコンセプト】
- アカウント名: ニートな管理栄養士｜AI丸投げ美肌ごはん
- 世界観: AI任せ、やる気ない、でも崩れない、美肌は維持、ゆるい
- 投稿の軸: AIにごはん決めさせる→そのまま食べる→管理栄養士的に軽く補足

【台本に含める要素】
1. 導入（やる気ない感）
2. AIが提案した献立
3. 実際の食事内容
4. 管理栄養士的補足（美肌効果など）
5. 締めの一言

【出力形式】
- 動画/画像の構成案
- キャプション文
- ハッシュタグ案
```

## 台本生成の流れ

1. このガイドを参照
2. Claudeに上記プロンプトを投げる
3. 生成された台本を `scripts/` に保存（001.md, 002.md...）
4. 必要に応じて編集
5. 実際に撮影・投稿

## テンプレート参照

投稿のフォーマットは `templates/post-template.md` を参照してください。

## 注意点

- 「ちゃんとしてないのに整ってる」を一貫して出す
- 栄養解説は軽く、専門的すぎない
- AI丸投げ感を常に演出
- 美肌要素を必ず入れる
