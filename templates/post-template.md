# 投稿テンプレート

## 基本フォーマット

### 導入
「今日もAIに丸投げ」「考えるのめんどくさい日」など、やる気ない感を出す

### メイン
AIが提案した献立を見せる
- 写真/動画で実際の食事を見せる
- 「これでいいらしい」というスタンス

### 補足
管理栄養士として軽く解説
- 栄養的にどうなのか
- 美肌にどう良いのか
- でもあくまで軽く、ゆるく

### 締め
「私は従うだけ」「ちゃんとしてないのに整ってる」を強調

---

## キャプション構成

```
【タイトル】
考えない日のごはん / AIに丸投げした / など

【本文】
- AI提案内容
- 実際に食べた感想
- 管理栄養士的補足（軽め）

【ハッシュタグ】
#美肌ごはん #AI活用 #管理栄養士 #ゆるい食生活 など
```
