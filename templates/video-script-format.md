# AI×管理栄養士ごはん｜動画構成（固定型）

## このファイルの使い方

このファイルは台本自動生成用のフォーマットです。

**ユーザーが入力:**
- 料理名や状況（例: 「無限キャベツの台本作って」「ケーキ食べた後のデトックス鍋」）

**AIが出力:**
1. タイムスタンプ付き完成台本（00:00〜00:28の具体的なセリフ・映像内容）
2. ChatGPT用プロンプト（スクショして使える形式）
3. ChatGPTからの想定回答（調理ポイント・栄養ワード含む）

## サムネイル

- フック画面を使用（00:00の完成ごはんドアップ or ChatGPT画面）
- ChatGPTロゴを入れる
- テキスト: 「AI×管理栄養士が作る ○○」（状況+料理名）

## 撮影ポイント

### 固定テロップ（毎回同じ）
1. **00:00, サムネ**: 「アラサーニート 美肌だけは死守したい AI丸投げ食生活」
2. **00:08**: 「指示通りやるだけ」
3. **00:10**: 「ここがポイントらしい」
4. **00:28**: 「レシピ詳細はコメント欄！」

### 固定画角（毎回同じアングル）
1. **00:04**: スマホ入力シーン（ChatGPT画面が見える）
2. **00:24**: いただきます（完成品を見せる）

### その他の固定撮影アングル
- **00:08**: 真上撮影（調理①）
- **00:10, 00:13**: 斜め撮影（調理②③）

### 演出ルール
- 固定で『アラサーニート（管理栄養士） 美肌だけは死守したい AI丸投げ食生活』の自己紹介
- 管理栄養士はアフレコでは読まない（テロップのみ）
- チャピに美肌・栄養意識した質問してレシピ出させる
- チャピの言った美肌ポイント、調理ポイントは動画内で実践して突っ込む
- 今日の様子、いただきます姿で女の子の影出す（顔出しはしない）
- 基本固定画角
- 完成時は無言で見せる
- 最後は軽めの感想で締める

## 可変要素（毎回変わる部分）

### 変わるテロップ・セリフ
1. **サムネ・00:00**: 状況+料理名（例: 「ケーキ後のデトックス鍋」「キャベツしかない時のサラダ」）
2. **00:04**: 今日の様子（例: 「ケーキパーティーしてきた」「冷蔵庫にキャベツしかない」）
3. **00:04**: ChatGPT画面の質問内容（例: 「ケーキ食べてむくんでる、何すればいい？」）
4. **00:06**: ChatGPTの提案内容（例: 「デトックス鍋なるものを提案された」）
5. **00:13**: 栄養ワード（例: 「ビタミンC」「食物繊維」「タンパク質」）
6. **00:26**: 感想（例: 「まあまあいいじゃん」「意外といける」）

### 変わる映像内容
- **00:00**: 完成ごはんのビジュアル（料理によって変わる）
- **00:04**: ChatGPT画面の内容
- **00:06**: ChatGPT回答の画面録画
- **00:08〜00:22**: 調理工程（料理によって変わる）
- **00:24**: 完成品

### フックパターン例（サムネ・00:00用）
状況で共感を引く:
- 金欠系: 「金欠納豆オムレツ」「100円で美肌ごはん」
- やる気ない系: 「やる気ゼロ3分丼」「考えたくない日のパスタ」
- 在庫消費系: 「納豆しかない時の○○」「冷蔵庫空っぽ救済ごはん」

### テーマパターン例（00:24用）
栄養・美肌効果で価値を伝える:
- 「ビタミンC美白」
- 「日焼けケア」
- 「腸活美肌」
- 「鉄分補給」
- 「タンパク質チャージ」

**構成**: フックで共感（自分ごと化）→ テーマで価値提供（保存理由）

反応が良かったパターンは備考欄にメモして、次回以降に活かす

## 節約・予算ネタ

金額や食費の話は別投稿/ストーリーズで展開:
- 「今週AI丸投げしたら食費○○円でした」
- 「納豆しかない時にAIに聞いてみた」
- プロンプトに予算を入れる例: 「納豆と卵しかありません、何作れる？」

## 台本生成時の出力ルール

台本作成依頼を受けたら、以下の形式で出力すること:

### 1. ChatGPT用プロンプト（スクショ用）
```
【ChatGPTに送る質問】
（状況に応じた具体的な質問文）
例: 「ケーキ食べてむくんでる、今日何すればいい？デトックス系で」
例: 「冷蔵庫にキャベツしかない、美肌に良いレシピ教えて」
```

### 2. ChatGPT想定回答
- 料理名
- 材料
- 作り方（調理ポイント含む）
- 美肌・栄養ワード（ビタミンC、デトックスなど）

### 3. 完成台本（タイムスタンプ付き）
上記のタイムライン構成に沿って、00:00〜00:28まで全て具体的に記載

**必須要素:**
- 固定フレーズ「アラサーニート 美肌だけは死守したい AI丸投げ食生活」
- 状況+料理名の組み合わせ
- ChatGPTの調理ポイント・栄養ワードを動画内で実践
- 管理栄養士ぽさを軽く出す（重くしない）
- 最後は「レシピ詳細はコメント欄！」で締め
